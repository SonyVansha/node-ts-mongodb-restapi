## Backend - API Setup

### Cloning the repository

```shell
git clone https://github.com/SonyVansha/node-ts-mongodb-restapi.git
```

### Install packages

```shell
npm i
```

### Setup MongoDB URL

In `src/index.ts`:

```js
const MONGO_URL = ''; // DB URI
```

### Start the app

```shell
npm start
```
